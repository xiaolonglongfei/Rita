## Critical Problem to Fix

The instructor profile page (`app/(main)/instructors/[id]/page.tsx`) is currently displaying:
- ❌ Reviewer's name (e.g. "Alex Chen", "Maria Santos")
- ❌ Review date/timestamp

**This is a serious privacy violation.** Rita guarantees complete anonymity to all reviewers. Student identity must NEVER be visible to instructors or any other user.

---

## Correct Review Display Rules

On the instructor profile page, each review card must show:

| Field | Show? | Notes |
|---|---|---|
| Student name | ❌ NEVER | Complete anonymity guaranteed |
| Student avatar | ❌ NEVER | Could help identify the reviewer |
| Review date | ❌ NEVER | Timing could help instructor identify reviewer |
| Verified badge | ✅ YES | Show if is_verified = true |
| Score (Value) | ✅ YES | rating_value |
| Score (Effectiveness) | ✅ YES | rating_effectiveness |
| Score (Punctuality) | ✅ YES | rating_punctuality |
| Overall score | ✅ YES | Average of three scores |
| Text comment | ✅ YES (if exists) | Only if comment is not null/empty |
| "Unverified" label | ✅ YES | Show if is_verified = false |

## Step 1: Update ReviewCard Component

Replace `components/review/ReviewCard.tsx` with the following:

```typescript
import { Badge } from '@/components/ui/badge'
import { Review } from '@/types/database'

interface ReviewCardProps {
  review: Review
}

export function ReviewCard({ review }: ReviewCardProps) {
  const overall = (
    (review.rating_value + review.rating_effectiveness + review.rating_punctuality) / 3
  ).toFixed(1)

  return (
    <div className="bg-white border border-slate-100 rounded-xl p-4">

      {/* Top row: overall score + verified badge */}
      <div className="flex items-center justify-between mb-3">
        <div className="flex items-center gap-2">
          {review.is_verified ? (
            <Badge className="bg-green-50 text-green-700 border-green-200 text-xs font-semibold">
              ✓ Verified Session
            </Badge>
          ) : (
            <Badge className="bg-slate-50 text-slate-500 border-slate-200 text-xs font-semibold">
              Unverified
            </Badge>
          )}
        </div>
        <span className="text-lg font-bold text-rita-blue">
          ⭐ {overall}
        </span>
      </div>

      {/* Three dimension scores */}
      <div className="flex gap-4 mb-3 text-xs font-semibold text-slate-500 uppercase tracking-wide">
        <span>💰 Value: <strong className="text-rita-blue">{review.rating_value.toFixed(1)}</strong></span>
        <span>📈 Effectiveness: <strong className="text-rita-blue">{review.rating_effectiveness.toFixed(1)}</strong></span>
        <span>⏰ Punctuality: <strong className="text-rita-blue">{review.rating_punctuality.toFixed(1)}</strong></span>
      </div>

      {/* Optional text comment */}
      {review.comment && review.comment.trim() !== '' && (
        <p className="text-sm text-slate-600 leading-relaxed border-l-2 border-rita-lime pl-3 mt-2">
          {review.comment}
        </p>
      )}

    </div>
  )
}
```
