## Fix 3: Review Cards — Score-Colored Left Border

In `components/review/ReviewCard.tsx`, replace any outline border with a left accent border that reflects the overall score:

```typescript
// Helper function — returns color based on overall score
function scoreColor(score: number): string {
  if (score >= 4.0) return '#1668c8'   // blue
  if (score >= 2.5) return '#c89000'   // amber
  return '#c83030'                      // red
}

// Calculate overall score
const overall = (
  (review.rating_value + review.rating_effectiveness + review.rating_punctuality) / 3
)

// Update the card container:
// REMOVE: any red border, outline border, or ring styling
// ❌ className="... border-red-500 ..."
// ❌ className="... ring-2 ring-red-400 ..."

// REPLACE WITH: white card + colored left border only
<div
  className="bg-white rounded-xl p-4 mb-3"
  style={{
    borderLeft: `4px solid ${scoreColor(overall)}`,
    borderTop: '1px solid #f1f5f9',
    borderRight: '1px solid #f1f5f9',
    borderBottom: '1px solid #f1f5f9',
  }}
>
  {/* card contents stay the same */}
</div>
```

This means:
- High scoring reviews (≥4.0) → blue left border
- Mid scoring reviews (2.5–3.9) → amber left border  
- Low scoring reviews (<2.5) → red left border
- All reviews have a subtle light gray border on the other three sides
